# DIO_Excel

Projeto: Dashboard em Excel para análise de dados (entrega DIO).

Descrição
--------
Este repositório contém um dashboard em Excel (.xlsx) com análises e visualizações finais. O objetivo é mostrar um exemplo de como dados tabulares podem ser transformados em um painel interativo usando recursos nativos do Excel (tabelas, tabelas dinâmicas, gráficos e formatações).

Arquivos
-------
- dashboard.xlsx: Arquivo Excel com dados e dashboard concluído.
- README.md: Este documento, com descrição do projeto e instruções para reprodução.

Dados utilizados
---------------
Os dados estão incluídos dentro do próprio arquivo `dashboard.xlsx` (planilha chamada "Dados" ou similar). Não há fontes externas externas necessárias para visualizar o dashboard; todos os dados e cálculos estão presentes no arquivo.

Requisitos
---------
- Microsoft Excel 2016, Excel 2019, Excel 2021 ou Microsoft 365 (recomendado) para suporte completo a tabelas dinâmicas e recursos de visualização.
- Alternativas: LibreOffice ou Google Sheets podem abrir o arquivo, mas algumas funcionalidades (gráficos avançados, segmentações, conexões e formatações) podem não ser totalmente compatíveis.

Como usar / reproduzir
---------------------
1. Clone ou baixe este repositório:
   git clone https://github.com/geancappellari/DIO_Excel.git

2. Abra o arquivo `dashboard.xlsx` com o Microsoft Excel.

3. Se o Excel perguntar sobre habilitar edição ou conteúdo, habilite a edição. Não há macros externas necessárias (se houver macros, o Excel irá solicitar a habilitação de macros explicitamente).

4. Para garantir que todas as análises estejam atualizadas, selecione Dados > Atualizar Tudo (Data > Refresh All).

5. Navegue pelas abas/planilhas do arquivo: você encontrará normalmente uma planilha de dados brutos (por exemplo, "Dados") e uma ou mais planilhas com o dashboard e gráficos.

Observações
---------
- Se o dashboard usar conexões a fontes externas (Power Query ou conexões de dados), será possível que o Excel solicite autenticação ao atualizar. Neste projeto, os dados principais estão embutidos, portanto isso não deve ocorrer.
- Caso alguma visualização não apareça corretamente, verifique a versão do Excel e tente abrir no Microsoft 365.
